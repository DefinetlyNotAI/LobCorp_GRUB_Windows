This directory contains the GRUB configuration files for the project. GRUB (GRand Unified Bootloader) 
is a bootloader package that supports multiple operating systems on a computer. 

The files in this directory are used to configure the GRUB bootloader 
to properly load the operating system(s) installed on the machine.

More information about how GRUB works and in details can be found [here](../docs/Boot%20-%20In%20detail.md).
You may also want to check the [GRUB documentation](https://www.gnu.org/software/grub/manual/grub/html_node/index.html)

The grub settings we are using is via Grub2Win, more information about it can be found [here](https://sourceforge.net/projects/grub2win/).
