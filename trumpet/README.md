This directory contains the source code for Trumpet, 
a Windows application that displays an overlay and plays a sound based on a "danger level" value.

The files in this directory include the C++ source code for the application, 
as well as the main media files, ico's and resources used by the application.

More information about how Trumpet works and in details can be found [here](../docs/Trumpet%20-%20In%20detail.md).
Basic information about the project can be found [here](../docs/Trumpet%20Info.md), this includes what most people need to know.
